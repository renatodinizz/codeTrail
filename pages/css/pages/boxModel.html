CSS

Modelo de Caixa

As propriedades de texto no CSS permitem controlar a aparência do texto em suas páginas web, desde a cor e o tamanho da fonte até o alinhamento e o espaçamento entre caracteres. Elas são essenciais para garantir a legibilidade e o apelo visual do conteúdo textual.

color

A propriedade color define a cor do texto de um elemento. Você pode especificar cores usando nomes de cores (por exemplo, red, blue), valores hexadecimais (por exemplo, #FF0000), valores RGB (por exemplo, rgb(255, 0, 0)) ou RGBA (por exemplo, rgba(255, 0, 0, 0.5) para transparência).

font-size

A propriedade font-size define o tamanho da fonte do texto. Você pode usar várias unidades para especificar o tamanho, como pixels(px), em, rem, porcentagens (%), entre outras. É recomendado usar unidades relativas como em ou rem para melhor responsividade.

Exemplo:

font-family

A propriedade font-family define a fonte a ser usada para o texto. Você pode especificar uma lista de fontes, separadas por vírgulas, como um "fallback" (alternativa). O navegador tentará carregar a primeira fonte da lista; se não estiver disponível, ele tentará a próxima, e assim por diante. É uma boa prática incluir uma família de fonte genérica (como serif, sans-serif, monospace).

font-weight

A propriedade font-weight define a espessura (negrito) da fonte. Os valores comuns incluem normal (equivalente a 400), bold (equivalente a 700), ou valores numéricos de 100 a 900 (em múltiplos de 100).

Exemplo:

line-height

A propriedade line-height define a altura de uma linha de texto, ou seja, o espaçamento vertical entre as linhas. Um valor sem unidade é geralmente o mais recomendado, pois é um fator multiplicador do font-size do elemento, o que ajuda na responsividade.

Exemplo:

text-align

A propriedade text-align define o alinhamento horizontal do texto dentro de um elemento de bloco. Os valores possíveis são left, right, center e justify.

Exemplo:

text-decoration

A propriedade text-decoration é usada para adicionar ou remover decorações de texto, como sublinhado, linha sobreposta ou tachado. Os valores comuns incluem none, underline, overline e line-through.

Exemplo:

text-transform

A propriedade text-transform controla a capitalização do texto. Os valores possíveis são none, uppercase (todas as letras maiúsculas), lowercase (todas as letras minúsculas) e capitalize (primeira letra de cada palavra maiúscula).

letter-spacing

A propriedade letter-spacing define o espaçamento horizontal entre os caracteres de texto. O valor pode ser em pixels (px), em, rem, etc. Valores positivos aumentam o espaçamento, e valores negativos o diminuem.

Exemplo:

word-spacing

A propriedade word-spacing define o espaçamento horizontal entre as palavras de texto. O valor pode ser em pixels (px), em, rem, etc.

Exemplo:

text-indent

A propriedade text-indent define a indentação da primeira linha de um bloco de texto. O valor pode ser em pixels (px), porcentagens (%), em, rem, etc. Valores negativos podem ser usados para criar um efeito de "hanging indent".
