CSS

Cores e Unidades

Cores

Existem várias maneiras de definir cores no CSS, cada uma com suas vantagens e casos de uso.

Cores por Nome

As cores podem ser especificadas usando nomes de cores predefinidos, como red, blue, green, black, white, etc. É a forma mais simples de usar cores, mas a variedade é limitada.

Exemplo:

p {
  color: blue;
}
h1 {
  background-color: lightgray;
}

Cores Hexadecimais

As cores hexadecimais são representadas por um # seguido de seis dígitos hexadecimais (0-9 e A-F). Os dois primeiros dígitos representam o valor de vermelho, os dois do meio o verde, e os dois últimos o azul. É uma forma compacta e muito comum de especificar cores.

Exemplo:

body {
  background-color: #F0F8FF;
}
h2 {
  color: #FF4500; /* OrangeRed */
}

Cores RGB (Red, Green, Blue)

As cores RGB são especificadas usando a função rgb(), que recebe três valores inteiros de 0 a 255, representando a intensidade de vermelho, verde e azul, respectivamente.

Exemplo:

.caixa {
  background-color: rgb(255, 99, 71);
}
.texto-rgb {
  color: rgb(0, 128, 0); /* Green */
}

Cores RGBA (Red, Green, Blue, Alpha)

As cores RGBA são uma extensão do RGB, adicionando um quarto valor, o canal alfa, que representa a opacidade do elemento. O valor alfa varia de 0 (totalmente transparente) a 1 (totalmente opaco).

Exemplo:

.overlay {
  background-color: rgba(0, 0, 0, 0.5);
  /* Preto com 50% de opacidade */
}

.texto-transparente {
  color: rgba(255, 0, 0, 0.7);
  /* Vermelho com 70% de opacidade */
}

Cores HSL ( )

As cores HSL são especificadas usando a função hsl(), que recebe três valores: matiz (hue), saturação (saturation) e luminosidade (lightness).

Matiz (Hue): Um grau no círculo de cores de 0 a 360 (0 = vermelho, 120 = verde, 240 = azul).
Saturação (Saturation): Uma porcentagem de 0% a 100% (0% = tom de cinza, 100% = cor pura).
Luminosidade (Lightness): Uma porcentagem de 0% a 100% (0% = preto, 100% = branco, 50% = cor normal).

Exemplo:

.botao {
  background-color: hsl(120, 100%, 50%);
}
.texto-hsl {
  color: hsl(240, 50%, 70%);
}

Cores HSLA ( )

Similar ao RGBA, o HSLA adiciona o canal alfa para controlar a opacidade, variando de 0 (totalmente transparente) a 1 (totalmente opaco).

Exemplo:

.sombra {
  background-color: hsla(0, 100%, 50%, 0.3);
  /* Vermelho com 30% de opacidade */
}

Unidades

As unidades CSS são usadas para especificar tamanhos, comprimentos, espaçamentos e outras dimensões. Elas podem ser absolutas ou relativas.

Unidades Fixas

Unidades absolutas são fixas e não mudam em relação a outros elementos ou ao tamanho da tela. São menos flexíveis para designs responsivos.

Pixels (px)

A unidade mais comum. Um pixel é um ponto na tela. É uma unidade absoluta, mas seu tamanho físico pode variar entre dispositivos.

Exemplo:

.largura-fixa {
  width: 200px;
}
.fonte-pequena {
  font-size: 12px;
}

Unidades Relativas

Unidades relativas são mais flexíveis e se ajustam em relação a outras medidas (como o tamanho da fonte do elemento pai, o tamanho da viewport, etc.). São ideais para criar designs responsivos.

porcentagem (%)

Relativa ao tamanho do elemento pai. Se um elemento pai tem 200px de largura, um filho com width:50% terá 100px.

Exemplo:

.container {
  width: 80%;
}
.imagem {
  width: 100%;
/* 100% da largura do container */
}

em

Relativa ao font-size do elemento pai. Se o elemento pai tem font-size: 16px, 1em será 16px. Se o elemento atual tiver seu próprio font-size definido, em será relativo a ele.

Exemplo:

.caixa {
  font-size: 16px;
  padding: 1em; /* 16px */
}
.texto-interno {
  font-size: 1.2em;
  /* 1.2 * 16px = 19.2px */
}

rem

Relativa ao font-size do elemento raiz <html>. Isso torna o dimensionamento mais previsível, pois não é afetado pelo font-size de elementos pais intermediários. Se o font-size do <html> for 16px, 1rem será sempre 16px.

Exemplo:

html {
  font-size: 16px;
}
h1 {
  font-size: 2.5rem;
  /* 2.5 * 16px = 40px */
}
p {
  font-size: 1rem;
  /* 1 * 16px = 16px */
}

vh (viewport height)

Relativa à altura da viewport (a janela visível do navegador). 1vh é igual a 1% da altura da viewport. 100vh preenche a altura total da tela.

Exemplo:

.secao-tela-cheia {
  height: 100vh;
}

vw (viewport width)

Relativa à largura da viewport. 1vw é igual a 1% da largura da viewport. 100vw preenche a largura total da tela.

Exemplo:

.titulo-responsivo {
  font-size: 5vw;
}

O tamanho da fonte se ajusta com a largura da tela.

vmin (viewport minimum)

Relativa à menor dimensão da viewport (altura ou largura). 1vmin é igual a 1% da menor dimensão da viewport.

Exemplo:

.circulo {
  width: 50vmin;
  height: 50vmin;
}

vmax (viewport maximum)

Relativa à maior dimensão da viewport (altura ou largura). 1vmax é igual a 1% da maior dimensão da viewport.

Exemplo:

.retangulo {
  width: 80vmax;
  height: 20vmax;
}

Em geral, é uma boa prática usar unidades relativas para a maioria dos elementos, especialmente para fontes e espaçamentos, isso ajuda a criar designs mais flexíveis e acessíveis que se adaptem bem a uma variedade de dispositivos e preferências do usuário.
