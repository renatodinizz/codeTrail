CSS

Seletores

Seletores CSS são padrões usados para selecionar os elementos HTML que você deseja estilizar.

Seletores Básicos

Seletor Global

Seleciona todos os elementos da página.

Exemplo: *

Seletor por Tag

Seleciona todos os elementos de um determinado tipo HTML. Por exemplo, se você quiser estilizar todos os títulos de nível 2 <h2>

Exemplo: h2

Seletor por ID

Seleciona um único elemento HTML com um atributo id específico. Um ID deve ser único em toda a página HTML. Ele é prefixado com um # (hash).

Exemplo: #meuId

Seletor por Classe

Seleciona todos os elementos HTML que possuem um atributo class específico. Ao contrário dos IDs, várias classes podem ser aplicadas a um único elemento, e a mesma classe pode ser usada em múltiplos elementos. Ele é prefixado com um . (ponto).

Exemplo: .minhaClasse

Combinadores

Seletor Descendente

O seletor descendente (espaço) seleciona todos os elementos que são descendentes de um elemento específico.

Exemplo: div p

Isso selecionará todos os parágrafos <p> que estão dentro de qualquer <div>.

Seletor Filho

O seletor filho > seleciona elementos que são filhos diretos de um elemento específico. Ele é mais restritivo que o seletor descendente.

Exemplo: ul > li

Isso selecionará apenas os itens de lista <li> que são filhos diretos de uma lista não ordenada <ul>.

Seletor Irmão Adjacente

O seletor irmão adjacente + seleciona um elemento que é imediatamente precedido por outro elemento específico, e ambos devem ter o mesmo pai.

Exemplo: h1 + p

Isso selecionará o primeiro parágrafo <p> que vem imediatamente após um título <h1>.

Seletor Irmão Geral

O seletor irmão geral ~ seleciona todos os elementos que são irmãos de um elemento específico e que o seguem na árvore do documento. Eles devem ter o mesmo pai.

Exemplo: p ~ span

Isso selecionará todos os elementos <span> que são irmãos de um parágrafo <p> e que aparecem depois dele.

Seletor por Propriedade

Seleciona os elementos com uma propriedade específica como name, type e etc.

Exemplo: input[type="password"]

Isso selecionará todos os elementos <inputs> que são do tipo password.

Agrupamento de Seletores

Você pode agrupar seletores para aplicar as mesmas regras de estilo a vários elementos sem repetir o código. Para agrupar seletores, separe-os com uma vírgula (,).

Exemplo: h1, h2, p { font-family: Arial, sans-serif;}

Neste exemplo, todos os <h1>, <h2> e <p> terão a mesma família de fonte.

Seletores Avançados

Podemos usar alguns seletores de forma mais avançadas, isso serve principalmente para quando não se tem acesso ao HTML da aplicação como no caso de usar uma ferramenta como WordPress ou outra parecida.

Para selecionar um elemento usamos um seletor avançado, vamos imaginar que queremos selecionar um elemento li que está em uma lista, para esse exemplo vamos usar o comando:

Ex: li:nth-child(3){} - Nesse exemplo o <li> é o item a ser selecionado, o n do nth vem de number e th da ordem cardial em inglês, child por que é um filho do elemento <ul> e o número dentro do parênteses está indicando que dentro da <ul> o terceiro <li> será selecionado.

Obs: Nesse caso consideramos o primeiro item como sendo o número 1 e não o 0 como em programação.

Ex: li:nth-child(2n){} - Nesse exemplo o n adicional depois do número significa que ele vai pular de dois em dois itens, ou seja, somente números pares, mas podemos substituir essa expressão pela palavra even dentro dos parênteses.

Ex: li:nth-child(2n-1){} - Nesse exemplo o 2n-1 depois do número significa que ele vai pular de dois em dois itens, mas começando pelo número 2-1 que é 1, ou seja, irá contar somente os números impares, mas podemos substituir essa expressão pela palavra odd dentro dos parênteses.

Outros Exemplos:

li:first-child - Seleciona o primeiro elemento filho.

li:last-child - Seleciona o último elemento filho.

li:not(first-child) - Seleciona todos os elementos menos o primeiro.

Especificidade de Seletores

A especificidade é um algoritmo que o navegador usa para determinar qual regra CSS aplicar a um elemento quando várias regras podem se aplicar.

A ordem de especificidade (do menos específico para o mais específico) é:

  1. Seletores Globais - Baixa especificidade
  2. Seletores de Tags - Baixa especificidade
  3. Seletores de Classe - Média especificidade
  4. Seletores de ID - Alta especificidade.
  5. Seletores Inline - Alta especificidade, pois são aplicados diretamente ao elemento.
  6. !important - Anula a especificidade normal. No entanto, seu uso é altamente desencorajado. Deve ser usada apenas como último recurso.
