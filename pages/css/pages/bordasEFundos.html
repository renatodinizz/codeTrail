Bordas e Fundos

Border

A propriedade border é uma shorthand para definir a largura, o estilo e a cor da borda de um elemento. Você pode controlar cada um desses aspectos individualmente.

border-width

Define a espessura da borda. Pode ser especificada em unidades como px, em, ou palavras-chave como thin, medium ou thick.

Exemplo:

div {
  border-width: 3px;
}
.caixa-grossa {
  border-width: thick;
}

border-style

Define o estilo da borda. É uma propriedade obrigatória para que a borda seja visível. Valores comuns incluem solid (linha contínua), dotted (pontilhada), dashed (tracejada), double (linha dupla), groove, ridge, inset, outset e none.

Exemplo:

p {
  border-style: solid;
}
.linha-tracejada {
  border-style: dashed;
}

border-color

Define a cor da borda. Pode ser especificada usando nomes de cores, valores hexadecimais, RGB, RGBA, HSL ou HSLA.

Exemplo:

h1 {
  border-color: blue;
}
.borda-vermelha {
  border-color: #FF0000;
}

Shorthand

.elemento {
  border: 1px solid black;
  /* largura estilo cor */
}

Você também pode definir bordas para lados específicos:

border-top, border-right, border-bottom, border-left, border-top-width, border-top-style, border-top-color, etc.

Border Radius

A propriedade border-radius permite arredondar os cantos de um elemento, criando formas mais suaves ou até mesmo círculos e elipses. Você pode aplicar o arredondamento a todos os cantos igualmente ou a cada canto individualmente.

div {
  border-radius: 12px;
  border-radius: 20px 0px;
  border-radius: 10px 5px 20px;
  border-radius: 5px 10px 15px 20px;
  border-radius: 10% / 50%;
}
  • 1 Valor: todos os cantos do elemento terão o mesmo raio de pixels.
  • 2 Valores: O primeiro é para top-left e bottom-right, segundo é para top-right e bottom-left.
  • 3 Valores: O primeiro é top-left, segundo é top-right e bottom-left, terceiro é bottom-right.
  • 4 Valores: São aplicados na ordem dos cantos, no sentido horário: top-left, top-right, bottom-right, bottom-left.
  • / : Quando usamos uma barra /, estamos definindo raios horizontais e verticais separadamente, criando cantos elípticos em vez de circulares.

Se quiser arredondar apenas um canto, use propriedades específicas:

div {
  border-top-left-radius: 8px;
  border-top-right-radius: 0;
  border-bottom-right-radius: 10px;
  border-bottom-left-radius: 5px;
}

Exemplos:

  • Exemplo 1: border-radius: none;
  • Exemplo 2: border-radius: 24px;
  • Exemplo 3: border-radius: 120px 30px;
  • Exemplo 4: border-radius: 50px 0 120px;
  • Exemplo 5: border-radius: 50%;
  • Exemplo 6: border-radius: 20% / 50%;
  • Exemplo 7: border-radius: 10% 100% / 120%;
  • Exemplo 8: border-radius: 50% 20% / 10% 40%;
  • Exemplo 9: border-radius: 70% 30% 30% 70% / 60% 40% 60% 40%;
  • Exemplo 10: border-radius: 50% 50% 0 50%;

Background

As propriedades de fundo controlam o preenchimento de um elemento com cores ou imagens.

background-color

Define a cor de fundo de um elemento. Pode ser qualquer valor de cor CSS (nome, hexadecimal, RGB, RGBA, HSL, HSLA).

Exemplo:

body {
  background-color: #f0f8ff;
}
.cabecalho {
  background-color: navy;
}

background-image

Define uma ou mais imagens de fundo para um elemento. Você pode especificar o caminho para a imagem usando a função url().

Exemplo:

.hero-section {
  background-image: url ("../img/images.jpg");
}

background-size

Controla o tamanho da imagem de fundo. Valores comuns incluem auto (tamanho original), cover (redimensiona a imagem para cobrir todo o elemento, cortando se necessário) e contain (redimensiona a imagem para caber dentro do elemento, sem cortar, mas pode deixar espaço vazio).

Exemplo:

.hero-section {
  background-image: url ("../img/images.jpg");
  background-size: cover;
}

background-position

Define a posição inicial da imagem de fundo. Pode ser especificada usando palavras-chave (por exemplo, center, top left), porcentagens ou unidades de comprimento (px, em).

Exemplo:

.hero-section {
  background-image: url ("../img/images.jpg");
  background-position: center center;
}
.logo {
  background-image: url ("../img/logo.jpg");
  background-position: 10px 20px;
}

background-repeat

Define se e como a imagem de fundo deve ser repetida. Valores comuns incluem repeat (padrão, repete horizontal e verticalmente), repeat-x (repete apenas horizontalmente), repeat-y (repete apenas verticalmente) e no-repeat (não repete).

Exemplo:

.padrao {
  background-image: url ("../img/images.jpg");
  background-repeat: repeat-x;
}
.unica-imagem {
  background-image: url ("../img/logo.jpg");
  background-repeat: no-repeat;
}

background-attachment

Define se a imagem de fundo rola com o conteúdo do elemento (scroll, padrão) ou se permanece fixa na viewport (fixed).

Exemplo:

body {
  background-image: url ("../img/logo.jpg");
  background-attachment: fixed;
  background-size: cover;
}

Shorthand

Para usar o shorthand basta usar somente a propriedade background, na ordem recomendada, ele combina:

  1. background-color - cor de fundo
  2. background-image - imagem de fundo
  3. background-position - posição da imagem
  4. background-size - (depois de uma /) → tamanho da imagem
  5. background-repeat - repetição da imagem
  6. background-origin - ponto de origem (bordas, padding, etc.)
  7. background-clip - até onde a cor/imagem se aplica
  8. background-attachment - rolagem fixa ou norma
