A tag <video> é uma ferramenta poderosa para enriquecer a experiência do usuário em suas páginas web, permitindo que você adicione vídeos relevantes ao conteúdo. Os formatos aceitos pelos navegadores são MP4, WebM e OGG.

O atributo controls adiciona controles de reprodução ao player de vídeo, permitindo que o usuário reproduza, pause e ajuste o volume do vídeo.

O atributo autoplay faz com que o vídeo comece a tocar automaticamente quando a página for carregada.

O atributo loop faz com que o vídeo seja reproduzido em loop, ou seja, quando chegar ao final, ele reiniciará automaticamente.

O atributo muted silencia o vídeo inicialmente, permitindo que o usuário ative o som manualmente.

O atributo preload indica ao navegador como deve carregar o vídeo. Os valores possíveis são none (não carregar), metadata (carregar apenas os metadados) e auto (carregar o vídeo completo). O mais recomendado é usar metadata para otimizar o desempenho da página.

O atributo poster permite definir uma imagem que será exibida antes do vídeo ser reproduzido. Isso é útil para fornecer uma prévia visual do conteúdo do vídeo.

O atributo width e height permitem definir as dimensões do player de vídeo. É importante especificar essas dimensões para garantir que o layout da página seja consistente.

O atributo crossorigin é usado para especificar como o navegador deve lidar com o carregamento de vídeos de domínios diferentes. Os valores possíveis são anonymous (sem credenciais) e use-credentials (com credenciais).

O atributo playsinline permite que o vídeo seja reproduzido inline em dispositivos móveis, em vez de abrir em tela cheia. Isso é especialmente útil para melhorar a experiência do usuário em dispositivos móveis.

O atributo disablepictureinpicture desativa o modo Picture-in-Picture, que permite que o vídeo seja reproduzido em uma janela flutuante enquanto o usuário navega por outras partes da página.

O atributo controlslist permite especificar quais controles devem ser exibidos no player de vídeo. Os valores possíveis incluem nodownload (oculta o botão de download), nofullscreen (oculta o botão de tela cheia) e noremoteplayback (oculta o botão de reprodução remota).

O atributo default indica que o vídeo deve ser reproduzido automaticamente quando a página for carregada, sem a necessidade de interação do usuário.

Video Remoto

Para usar vídeos remotos, como os hospedados no YouTube, por exemplo. Para isso, você pode usar a tag <video> com o atributo src apontando para a URL do vídeo remoto.

Outro método é usar a tag <iframe> para incorporar vídeos de plataformas como YouTube, Vimeo, etc. Isso permite que você exiba vídeos diretamente em sua página sem precisar hospedar os arquivos localmente. E é a maneira mais recomendada para vídeos do YouTube, pois eles fornecem um player otimizado e recursos adicionais.

Para usar um video remoto do Youtube, você pode seguir os passos abaixo:

  1. Entre em youtube.com
  2. Escolha o video que deseja
  3. Clique em compartilhar
  4. Clique em incorporar
  5. Copie o código
  6. Cole o código onde desejar
