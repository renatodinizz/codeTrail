Edição de Texto

Negrito

O elemento <strong> é usado para destacar o texto com ênfase forte, geralmente exibido em negrito. Por exemplo, este texto está em negrito.

O elemento <b> também é usado para destacar o texto com negrito, porém está em desuso.

Itálico

O elemento <em> é usado para enfatizar o texto, geralmente exibido em itálico. Por exemplo, este texto está em itálico.

O elemento <i> também é usado para destacar o texto com itálico, porém está em desuso.

Marcação

O elemento <mark> é usado para destacar o texto que é relevante ou importante, geralmente exibido com fundo amarelo. Por exemplo, este texto está destacado.

Com CSS é possível estilizar o mark para ter uma cor de fundo diferente, como por exemplo: este texto está destacado com fundo azul.

Diminuir Texto

O elemento <small> é usado para exibir texto em tamanho menor, geralmente usado para notas de rodapé ou informações secundárias. Por exemplo, este texto está em tamanho menor.

Riscar o Texto

O elemento <del> é usado para indicar que o texto foi removido ou excluído, geralmente exibido com uma linha sobre o texto. Por exemplo, este texto foi removido.

O elemento <s> também é usado para riscar o texto, porém está em desuso.

Sublinhar o Texto

O elemento <ins> é usado para indicar que o texto foi adicionado, geralmente exibido com uma linha sublinhada. Por exemplo, este texto foi adicionado.

O elemento <u> também é usado para sublinhar o texto, porém está em desuso.

Texto Invertido

O elemento <bdo> é usado para alterar a direção do texto, seja da esquerda para a direita (LTR) ou da direita para a esquerda (RTL). Por exemplo, Este texto está da direita para a esquerda.

Precisa colocar o atributo dir com o valor rtl, que significa right to left, para inverter a direção do texto.

Sobrescrever Texto

o elemento sub <sup> é usado para exibir texto como sobrescrito, geralmente usado em notações matemáticas ou científicas. Por exemplo, E=mc2 representa a famosa equação de Einstein.

Neste exemplo, o texto dentro do elemento <sup> é exibido acima da linha de base do texto normal, enquanto o texto fora do elemento <sup> é exibido na linha da base.

Subscrever Texto

O elemento <sub> é usado para exibir texto como subscrito, geralmente usado em notações matemáticas ou científicas. Por exemplo, H2O representa a água.

Neste exemplo, o texto dentro do elemento <sub> é exibido abaixo da linha de base do texto normal, enquanto o texto fora do elemento <sup> é exibido na linha de base.

Entrada do Teclado

O elemento <kbd> é usado para representar a entrada do teclado, geralmente exibido com uma fonte monoespaçada. Por exemplo, Ctrl + Alt + Del é uma combinação comum usada para acessar o gerenciador de tarefas em sistemas operacionais.

Time

O elemento <time> é usado para representar uma data ou hora, geralmente exibido com um atributo datetime que fornece a data/hora em formato ISO.

O atributo datetime é importante para fornecer informações semânticas sobre a data/hora, permitindo que os navegadores e mecanismos de busca entendam melhor o conteúdo.

No atributo datetime, o formato é YYYY-MM-DDTHH:MM:SS, onde:

  • YYYY é o ano (4 dígitos)
  • MM é o mês (2 dígitos, de 01 a 12)
  • DD é o dia do mês (2 dígitos, de 01 a 31)
  • T é um separador que indica o início da parte de tempo
  • HH é a hora (2 dígitos, de 00 a 23)
  • MM é os minutos (2 dígitos, de 00 a 59)
  • SS é os segundos (2 dígitos, de 00 a 59)

Por exemplo, a data... 1 de outubro de 2023 às 10:00 representa uma data e hora específicas. Que deve ser exibido no atributo datetime, como 2023-10-01T10:00:00.

Endereço

O elemento <address> é útil para fornecer informações de contato em páginas da web, como rodapés ou seções de contato.

O elemento <address> é usado para representar informações de contato, geralmente exibido em itálico. Pode conter informações como nome, endereço, telefone e e-mail. E tem comportamento de uma estrutura em bloco, ou seja, quebra uma linha antes e depois do conteúdo. Por exemplo,

Rua Exemplo, 123, Cidade, País
representa um endereço físico.

Outro exemplo,

João da Silva, Rua Exemplo, 123, Cidade, País, (11) 1234-5678,
