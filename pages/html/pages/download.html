Download

Disponibilizar um arquivo para download significa permitir que o usuário baixe um arquivo diretamente ao clicar em um link. Pode ser um PDF, um documento Word, planilha, imagem, áudio, vídeo, ZIP etc.

Download Direto

Para disponibilizar um arquivo para download, você pode usar a tag <a> com o atributo download . Isso permite que os usuários baixem o arquivo em vez de abri-lo diretamente no navegador.

Exemplo de uso:

Para baixar o PDF clique aqui

Estrutura do HTML:

O Atributo href indica o caminho ou URL do arquivo que será baixado. O Atributo download indica ao navegador que deve baixar o arquivo (não apenas abrir). O atributo type indica o tipo do arquivo.

Observações importantes: O atributo download não funciona com arquivos hospedados em outros domínios se o servidor não permitir (ex: Google Drive).

Visualização Prévia

Se você quiser que o usuário visualize o arquivo antes de baixá-lo, pode simplesmente usar a tag <a> com o atributo href apontando para o arquivo e o atributo target com o valor _blank. Isso abrirá o arquivo em uma nova aba do navegador.

Exemplo de uso:

Para baixar o PDF clique aqui

Estrutura do HTML:

O Atributo href indica o caminho ou URL do arquivo que será baixado. O Atributo download indica ao navegador que deve baixar o arquivo (não apenas abrir). O atributo type indica o tipo do arquivo.

Obs: Se você quer que o PDF abra em uma nova aba (pré-visualização) em vez de baixar direto, não use o atributo download. O atributo download força o navegador a baixar o arquivo e ignora target="_blank".

Nomear Arquivo

Se você quiser dar um nome personalizado ao arquivo que será baixado, pode usar o atributo download com um valor. Isso permite que você especifique o nome do arquivo quando ele for baixado.
