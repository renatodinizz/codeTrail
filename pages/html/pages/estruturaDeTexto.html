Estrutura de Texto em HTML

Títulos

Os títulos são definidos pelas tags <h1> a <h6>, onde <h1> é o mais importante e <h6> o menos importante. Eles são usados para estruturar o conteúdo e melhorar a acessibilidade.

Não se deve usar títulos para estilização de texto, mas sim para indicar a hierarquia do conteúdo. Para estilizar o texto, use CSS.

Parágrafos

Os parágrafos são definidos pela tag <p>. Eles são usados para agrupar blocos de texto e devem ser usados para separar ideias ou seções do conteúdo.

É importante usar parágrafos para melhorar a legibilidade do texto e facilitar a compreensão do conteúdo.

Pode se usar o comando lorem ipsum para preencher o texto durante o desenvolvimento. um exemplo de um parágrafo com lorem ipsum seria:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Deserunt maiores velit vero deleniti temporibus, dicta dolorum debitis consequatur obcaecati veniam in dolorem eos dignissimos! Molestias non natus minima adipisci nulla!

Quebra de linha

A quebra de linha é feita com a tag <br>. Ela é usada para separar linhas de texto sem criar um novo parágrafo. É útil em casos como endereços, poesias ou listas curtas. No entanto, deve ser usada com moderação, pois o uso excessivo pode prejudicar a legibilidade do texto. E não deve ser usada para espaçamento entre parágrafos, pois isso deve ser feito com CSS.

Outro exemplo de quebra de linha é a tag <hr>, que cria uma linha horizontal para separar seções do conteúdo. Ela é usada para indicar uma mudança de tópico ou seção, mas não deve ser usada como um substituto para parágrafos ou títulos.

Citações de Títulos

As citações de títulos são definidas pela tag <cite>. Elas são usadas para indicar o título de uma obra, como livros, filmes, artigos ou sites. A tag <cite> deve ser usada para títulos de obras literárias, artísticas ou científicas. Geralmente, o texto dentro da tag <cite> é exibido em itálico.

Por exemplo, O Pequeno Príncipe é um livro famoso escrito por Antoine de Saint-Exupéry.

Citações simples

As citações simples são definidas pela tag <q>. Elas são usadas para indicar que um trecho de texto é uma citação direta de outra fonte. A tag <q> deve ser usada para citações curtas, que podem ser incluídas dentro de um parágrafo. Isso ajuda a melhorar a acessibilidade do conteúdo e a indicar que o texto é uma citação.

Exemplo de citação simples.

O pai de um amigo meu já dizia: O computador é um burro muito rápido

Nesse exemplo, só o fato de usar a tag <q> já indica que o texto é uma citação direta e automaticamente é inserida as aspas.

Citações grandes

As citações grandes são definidas pela tag <blockquote>. Elas são usadas para indicar que um trecho de texto é uma citação longa ou complexa, que deve ser destacada do restante do conteúdo. A tag <blockquote> deve ser usada para citações que ocupam mais de uma linha e geralmente são formatadas com recuo.

Abreviações

As abreviações são definidas pela tag <abbr>. Elas são usadas para indicar que um termo é uma abreviação ou sigla. A tag <abbr> deve ser usada para termos que são comumente abreviados, como nomes de organizações, instituições ou termos técnicos. Ao deixar o mouse sobre a abreviação, o texto completo da abreviação é exibido, o que ajuda a melhorar a compreensão do conteúdo.

Para usar, você deve incluir no atributo title o texto completo da abreviação, e dentro da tag <abbr> o termo ou a sigla. Isso ajuda a melhorar a acessibilidade do conteúdo, pois os leitores de tela podem ler o texto completo da abreviação para pessoas com deficiência visual.

Exibindo Código no HTML

Quando você precisa exibir código ou texto que deve manter sua formatação original, é importante usar as tags apropriadas para garantir que o texto seja exibido corretamente. Isso é especialmente útil quando você está escrevendo documentação, tutoriais ou qualquer outro conteúdo que envolva código.

Para exibir o texto como um código, em uma fonte de largura fixa, você pode usar a tag <pre> ou a tag <code>. A tag <pre> preserva os espaços em branco e quebras de linha, enquanto a tag <code> é usada para indicar que o texto é código.

code

A tag <code> é usada para exibir código de programação ou trechos de código. Ela não preserva espaços em branco e quebras de linha, mas indica que o texto é código.

Exemplo sem o uso da tag <code>:

console.log('Hello, World!');

Exemplo com o uso da tag <code>:

console.log('Hello, World!');

pre

A tag <pre> é usada para exibir texto pré-formatado, ou seja, exibe o texto exatamente como ele foi escrito, preservando espaços em branco e quebras de linha. É útil para exibir código ou texto que precisa manter sua formatação original.

Exemplo de uso da tag <pre>: 

                        
                            num = int(imput("digite um número"))
                            if num % 2 == 0:
                                print(f "o número(num) é PAR")
                            else:
                                print(f "o número(num) é IMPAR")
                            print("fim do projeto")

Nesse exemplo, o código é exibido exatamente como foi escrito, com a formatação preservada. O código está dentro da tag <code>, que está dentro da tag <pre>.

Para deixar claro, vamos ver o mesmo trecho de código sem o uso da tag <pre>:

num = int(imput("digite um número")) if num % 2 == 0: print(f "o número(num) é PAR") else: print(f "o número(num) é IMPAR") print("fim do projeto")

Div

A tag <div> é usada para agrupar elementos HTML em um bloco, ou seja, uma div ocupa todo o espaço horizontal disponível na tela. Ela não tem significado semântico, mas é útil para aplicar estilos CSS ou scripts JavaScript a um grupo de elementos. É uma das tags mais comuns usadas para estruturar o layout de uma página web.

Ela pode conter outros elementos HTML, como parágrafos, imagens, listas, etc. É importante lembrar que a tag <div> não deve ser usada para substituir outras tags semânticas, como <header>, <footer>, <article>, etc.

Span

A tag <span> é usada para agrupar elementos HTML em linha. Ela não tem significado semântico, mas é útil para aplicar estilos CSS ou scripts JavaScript a um grupo de elementos em linha.

A tag <span> é útil quando você precisa aplicar estilos a uma parte específica de um texto, mas não quer criar um novo bloco de texto. Por exemplo, você pode usar a tag <span> para destacar uma palavra ou frase dentro de um parágrafo.
