HTML

Tags

As tags são os elementos fundamentais do HTML, usadas para estruturar e formatar o conteúdo de uma página web. Cada tag tem um propósito específico e pode conter atributos que modificam seu comportamento ou aparência. Aqui estão algumas das tags mais comuns:

Estrutura Básica

  • <!DOCTYPE html> - Declara o tipo do documento como HTML5, deve ser o primeiro elemento
  • <html> - Elemento raiz que engloba todo o conteúdo
  • <head> - Contém metadados, links para CSS, scripts, etc.
  • <title> - Define o título que aparece na aba/navegador
  • <body> - Contém todo o conteúdo visível da página

Conteúdo e Formatação

  • <h1> a <h6> - Títulos de diferentes níveis semânticos
  • <p> - Parágrafo de texto
  • <br> - Quebra de linha simples (elemento vazio)
  • <hr> - Linha horizontal indicativa de separação temática
  • <!-- comentário --> - Comentário visível somente no código-fonte
  • <strong> - Texto de importância (negrito) conforme significado semântico.
  • <em> - Texto de ênfase (itálico) conforme significado semântico

Blocos e Seções Semânticas

  • <div> - Bloco genérico de divisão sem semântica específica
  • <span> - Contêiner inline genérico para estilo ou agrupamento
  • <header> - Cabeçalho de uma seção ou do documento
  • <footer> - Rodapé de uma seção ou do documento
  • <main> - Conteúdo principal da página de forma semântica
  • <section> - Seção temática do documento
  • <article> - Conteúdo independente (ex. postagem, notícia)
  • <aside> - Conteúdo complementar ou contextual

Listas

  • <ul> - Lista não ordenada
  • <ol> - Lista ordenada
  • <li> - Item de lista
  • <dl>, <dt> e <dd> - lista de definições (termos e suas descrições)

Imagens, Mídia e Multimídia

  • <img> - Insere imagem (atributo src)
  • <picture> - Contêiner para diferentes fontes de imagem (resolução)
  • <figure> e <figcaption> - Figura com legenda associada
  • <audio> - Encaixam com conteúdo de som
  • <video> - Encaixam com conteúdo de vídeo
  • <source> - Define múltiplas fontes de mídia para <audio> e <video>
  • <track> - Legendas ou trilhas de texto para mídia audiovisual

Formulários e Entradas

  • <form> - Formulário para submissão de dados
  • <input> - Campo de entrada (texto, email, número, etc.)
  • <textarea> - Campo de texto multilinha
  • <button> - Botão clicável
  • <select>, <option> e <optgroup> - Lista suspensa com opções agrupadas
  • <label> - Etiqueta associada a um <input>
  • <fieldset> e <legend> - Agrupam controles de formulário com legenda
  • <datalist> - Lista de sugestões para <input>
  • <output> - Apresenta o resultado de um cálculo ou ação do usuário

Semântica Avançada e Dados

  • <mark> - Destaca texto de interesse
  • <time> - Marca data/hora de forma legível e legível por máquina
  • <data> - Conecta dados legíveis ao conteúdo humano
  • <progress> e <meter> - Indicadores visuais de progresso ou valor numérico

Scripts, Estilos e Metadados

  • <link> - Vincula recursos externos como CSS e ícones
  • <meta> - Metadados, como charset, viewport, SEO
  • <style> - Define estilos CSS embutidos
  • <script> - Inclui ou referencia códigos JavaScript
  • <noscript> - Conteúdo a ser exibido se scripts estiverem desativados

Outros Elementos Diversos

  • <object>, <embed> e <param> - Incorporam conteúdos externos ou plugins
  • <canvas> - Permite desenhos dinâmicos via JavaScript
  • <svg> - Conteúdo vetorial escalável diretamente no HTML
  • <details> e <summary> - Painéis de descrição expansível
  • <dialog> - Define uma janela/modal interativa
  • <template> - Modelo de conteúdo que permanece oculto até ser usado via script
